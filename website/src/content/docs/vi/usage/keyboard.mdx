Thanh công cụ bàn phím cung cấp các nút thân thiện cảm ứng cho các phím terminal phổ biến.

## Nhóm nút

### Bộ điều chỉnh

| Nút   | Phím        | Trường hợp              |
| ----- | ----------- | ----------------------- |
| Tab   | Tab         | Tự động hoàn thành      |
| Esc   | Escape      | Thoát chế độ, hủy       |
| Enter | Enter       | Gửi lệnh                |
| Ctrl  | Ctrl (dính) | Bộ điều chỉnh cho combo |

### Phím mũi tên

| Nút | Phím       |
| --- | ---------- |
| ↑   | Phím lên   |
| ↓   | Phím xuống |
| ←   | Phím trái  |
| →   | Phím phải  |

### Combo Ctrl

| Nút | Combo  | Trường hợp   |
| --- | ------ | ------------ |
| C   | Ctrl+C | Hủy/ngắt     |
| D   | Ctrl+D | EOF/logout   |
| Z   | Ctrl+Z | Tạm dừng     |
| L   | Ctrl+L | Xóa màn hình |
| A   | Ctrl+A | Đầu dòng     |
| E   | Ctrl+E | Cuối dòng    |
| U   | Ctrl+U | Xóa dòng     |
| K   | Ctrl+K | Xóa đến cuối |

### Điều khiển cuộn

| Nút  | Hành động               |
| ---- | ----------------------- |
| PgUp | Trang lên               |
| PgDn | Trang xuống             |
| Copy | Bật/tắt chế độ sao chép |

## Mục tiêu cảm ứng

- Kích thước tối thiểu: 44x44px
- Khoảng cách: 8px giữa các nút
- Phản hồi rung khi chạm (nếu thiết bị hỗ trợ)

## Mẹo

1. **Ctrl dính**: Chạm một lần, sau đó chạm phím khác
2. **Chế độ sao chép**: Bật để chọn văn bản, tắt để tiếp tục gõ
3. **Xóa nhanh**: Chạm L (Ctrl+L) để xóa màn hình
