title: Keyboard Toolbar
description: Virtual keyboard for mobile terminal control

The keyboard toolbar provides touch-friendly buttons for common terminal keys.

## Button Groups

### Modifiers

| Button | Key           | Use Case            |
| ------ | ------------- | ------------------- |
| Tab    | Tab           | Autocomplete        |
| Esc    | Escape        | Exit modes, cancel  |
| Enter  | Enter         | Submit command      |
| Ctrl   | Ctrl (sticky) | Modifier for combos |

### Arrow Keys

| Button | Key         |
| ------ | ----------- |
| up     | Up Arrow    |
| down   | Down Arrow  |
| left   | Left Arrow  |
| right  | Right Arrow |

### Ctrl Combos

| Button | Combo  | Use Case         |
| ------ | ------ | ---------------- |
| C      | Ctrl+C | Cancel/interrupt |
| D      | Ctrl+D | EOF/logout       |
| Z      | Ctrl+Z | Suspend          |
| L      | Ctrl+L | Clear screen     |
| A      | Ctrl+A | Line start       |
| E      | Ctrl+E | Line end         |
| U      | Ctrl+U | Clear line       |
| K      | Ctrl+K | Delete to end    |

### Scroll Controls

| Button | Action           |
| ------ | ---------------- |
| PgUp   | Page up          |
| PgDn   | Page down        |
| Copy   | Toggle copy mode |

## Touch Targets

- Minimum size: 44x44px
- Spacing: 8px between buttons
- Haptic feedback on tap (if device supports)

## Tips

1. **Ctrl is sticky**: Tap once, then tap another key
2. **Copy mode**: Enable to select text, disable to resume typing
3. **Quick clear**: Tap L (Ctrl+L) to clear screen
