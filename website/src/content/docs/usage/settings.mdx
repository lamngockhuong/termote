---
title: Settings
description: Configure Termote preferences
---

Access settings via the **gear icon** (⚙) in the header → **Preferences**.

## Text Input Send Behavior

Controls what happens when you tap **Send** in the text input mode (non-Latin input).

| Option         | Behavior                                      | Default |
| -------------- | --------------------------------------------- | ------- |
| Send text only | Sends text to terminal, no Enter key appended | ✓       |
| Send + Enter   | Sends text then automatically presses Enter   |         |

**Use case**: Set to "Send + Enter" if you want to type a command in Vietnamese/CJK and submit it in one tap.

## Paste Button Source

Controls where the **Paste** button in the toolbar gets content from.

| Option           | Behavior                                        | Default |
| ---------------- | ----------------------------------------------- | ------- |
| System clipboard | Paste from device clipboard (Ctrl+Shift+V)      | ✓       |
| tmux buffer      | Paste from tmux copy mode buffer (Ctrl+B, ])    |         |

**Tip**: Use "System clipboard" for pasting from other apps. Use "tmux buffer" for pasting text copied within tmux.

## Toolbar Default Expanded

Controls whether the keyboard toolbar starts in expanded or minimal mode when the page loads.

| Option | Behavior                                    | Default |
| ------ | ------------------------------------------- | ------- |
| Off    | Toolbar starts in minimal mode (fewer keys) | ✓       |
| On     | Toolbar starts expanded (all keys visible)  |         |

## Disable Right-Click Menu

Blocks the browser context menu (right-click / long-press) on the terminal area to prevent accidental interruptions.

| Option | Behavior                                | Default |
| ------ | --------------------------------------- | ------- |
| On     | Context menu is blocked on terminal     | ✓       |
| Off    | Browser default context menu is allowed |         |

## Show Session Tabs (Desktop Only)

Displays a horizontal tab bar at the top for quick session switching on desktop. The sidebar remains available for more detailed session management.

| Option | Behavior                              | Default |
| ------ | ------------------------------------- | ------- |
| On     | Show session tabs in header           | ✓       |
| Off    | Hide tabs, use sidebar only           |         |

This setting only appears on desktop devices.

## Session Poll Interval

Controls how often the app syncs the session list from the server. Lower values keep multi-device views in sync faster; higher values reduce server load when using a single device.

| Option                            | Default |
| --------------------------------- | ------- |
| 3s, 5s, 10s, 15s, 30s, 1m, 2m, 5m | 5s ✓    |

## Check for Updates

Manually check GitHub for new Termote releases. Shows inline result (already up to date, new version available, or error message). The app also auto-checks on load.

## Clear Command History

Deletes all saved commands from the command history dropdown. Use when you want to free up stored history or start fresh.

## Show Gesture Hints (Mobile Only)

Re-opens the gesture tutorial overlay that appeared on your first visit. Useful if you want to review available touch gestures.

This button only appears on mobile devices.

## Persistence

All settings are saved to your browser's local storage (`termote-settings` key) and persist across sessions. Clearing browser data or using "Clear Cache & Reload" will reset settings to defaults.
